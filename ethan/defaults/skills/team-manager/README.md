# Team Manager

一站式团队管理技能，覆盖三大核心场景：**绩效管理**、**任务委派**、**时间线驱动**。

## 适用人群

管理 5-20 人技术团队的 leader / 管理者，希望 Agent 辅助日常管理、减少重复跟进工作。

## 三大场景

### 1. 绩效管理

平时持续收集成员工作事件，绩效季自动汇总并生成评估建议。

**信息来源**：
- 手动记录 — 随时跟 Agent 说，自动归类存储
- 文档分析 — 转发下属文档时，提取亮点数据、架构思路、业务价值
- CR 统计 — 按人汇总代码产出、质量问题、需求归属
- 群消息提取 — 通过定时任务扫描指定群，识别候选事件

### 2. 任务委派

将任务分配给成员，自动拆解阶段、创建飞书任务、设置 checkpoint 提醒。

**流程**：用户描述任务 → Agent 拆分阶段 → 确认 DDL → 创建飞书任务 → 关键节点自动提醒

### 3. 时间线驱动

管理绩效周期、OKR、季度汇报等周期性事件，在关键节点自动触发提醒和操作。

**特点**：
- 声明式配置（anchor + offset + recurrence），不穷举年份
- Engine 运行时计算当前阶段，自动编译为定时任务
- 支持导出 / 导入备份

## 快速开始

### 1. 安装

重启 Ethan Agent 即可，技能会自动同步到 `~/.ethan/skills/`，`~/.ethan/work/` 目录及配置文件也会自动创建。

### 2. 配置团队

编辑 `~/.ethan/work/team.yaml`，填入你的团队信息：

```yaml
self:
  name: "张三"
  role: "前端负责人"

members:
  - name: "李四"
    role: "高级工程师"
    direction: "基础架构"
    joined: "2024-03"
    code_username: "lisi"
```

### 3. 配置时间线（可选）

编辑 `~/.ethan/work/timelines.yaml`，或直接告诉 Agent：

> 「每年 7 月开始半年绩效，1 月开始全年绩效」

Agent 会自动解析并生成配置。

### 4. 开始使用

直接用自然语言触发：

| 场景 | 示例 |
|---|---|
| 记录事件 | 「记一下，小王今天主动排查了线上问题，10 分钟定位到根因」 |
| 文档分析 | 「帮我看看这个文档」+ 转发链接 |
| CR 统计 | 「帮我看下这周的 CR 汇总」 |
| 任务委派 | 「安排一下小王，设计插件架构，下周五前出方案」 |
| 群监控 | 「帮我监控这个群，每天扫一次」 |
| 时间线 | 「现在是什么阶段」「下一个节点是什么」 |
| 绩效汇总 | 「帮我汇总下这季度的绩效」 |
| 导出备份 | 「导出时间线」 |

## 数据存储

所有数据存储在本地 `~/.ethan/work/`：

```
~/.ethan/work/
├── team.yaml                # 团队配置
├── timelines.yaml           # 时间线配置
├── .timeline_state.json     # 时间线运行状态
├── people/                  # 人员日志（唯一记录源）
│   ├── 李四.md
│   └── _self.md
├── cr-reports/              # CR 分析报告
├── reviews/                 # 绩效评估草稿
└── exports/                 # 导出文件
```

## 定时任务分类

| 类别 | 说明 | 示例 |
|---|---|---|
| 一次性 `one_off` | 执行一次后删除 | 提醒小王明天交方案 |
| 周期性 `recurring` | 固定频率重复 | 每周五发 CR 周报 |
| 时间线 `timeline` | 由时间线阶段驱动 | 绩效季前 2 周汇总提醒 |

## 场景标签

所有任务和提醒都带场景标签，互不干扰：

`work` / `life` / `health` / `study` / `finance` / `social`

Agent 会根据内容自动推断标签，无法推断时会询问。

## 设计原则

- **不做最终判定**：Agent 只提供汇总和建议，绩效打分由管理者决定
- **确认后再创建**：任务拆分、飞书任务创建，必须经用户确认
- **隐私边界**：所有数据仅存本地，不上传外部
- **通用框架**：不含任何个人/公司/团队信息，开箱即用
