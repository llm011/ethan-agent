# Obsidian Vault Manager

Ported from [NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent/tree/main/skills/note-taking/obsidian) (MIT License).

## Overview

Filesystem-first Obsidian vault management skill. Supports reading, searching, creating, editing notes, and adding wikilinks.

## Usage

Set `OBSIDIAN_VAULT_PATH` environment variable to your vault path, or it defaults to `~/Documents/Obsidian Vault`.
